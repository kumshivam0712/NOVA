# BrandForge Complete

AI content platform for Indian creators — full hackathon project package.

## Project location

All source code lives in `brandforge-full/`:

```
BrandForge_Complete/
└── brandforge-full/
    ├── backend/     # Express API (port 3001)
    └── frontend/    # React app (port 3000)
```

## Quick start

```bash
cd brandforge-full/backend
copy .env.example .env   # add your API key
npm install
npm start

# New terminal
cd brandforge-full/frontend
npm install
npm start
```

Open **http://localhost:3000**
