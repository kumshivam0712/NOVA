### The AI Content Team for India's 80M+ One-Person Brands

**Built at iQOO Hackathon 2026 by Team Nova in just 8 hours.**

---

## 🌟 Overview

Millions of Indian creators, solopreneurs, home businesses, and local sellers struggle to create engaging content consistently. Most spend hours every day writing captions, planning posts, and creating marketing content across multiple platforms.

**BrandForge Nova** transforms a single idea into complete social media content in seconds.

Simply speak or type your idea in Hindi, Hinglish, Tamil, Bengali, or other Indian languages, and Nova instantly generates:

- 🎬 Reels Scripts
- 🖼️ Instagram Carousels
- 🧵 X (Twitter) Threads

All tailored to your unique brand voice.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Expo CLI
- Anthropic API Key

### Installation

```bash
git clone https://github.com/kumshivam0712/NOVA.git

cd NOVA

npm install
```

### Environment Setup

Create a `.env.local` file:

```env
ANTHROPIC_API_KEY=your_api_key_here
```

> ⚠️ Never commit API keys to GitHub.

### Run Locally

```bash
npm run dev
```

For Expo:

```bash
npx expo start
```
